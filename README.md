# Flowers Cards

A simple static web page that displays a grid of flower image cards generated with JavaScript.

**Preview**
- Opens a responsive collection of small cards showing flower images.

**How to run**
- Open [index.html](index.html) in a browser (double-click or use a local server).
- Or use VS Code Live Server extension to serve the folder.

**Files**
- [index.html](index.html) — main HTML file
- [script.js](script.js) — generates the grid of image cards
- [style.css](style.css) — basic layout and card styling

**Notes**
- Images are loaded from remote URLs; an internet connection is required.
- `script.js` creates multiple cards by choosing random images from a preset list.

**Customize**
- Edit the image URLs array in `script.js` to use your own images.
- Modify sizes and layout in `style.css`.
